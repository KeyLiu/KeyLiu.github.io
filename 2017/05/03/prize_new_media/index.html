新媒体下的“中奖”教程

发表于 2017-05-03 | 分类于 新媒体运营 | 阅读次数

看到这个标题，是不是感觉十分荒谬。其实，当我准备写这篇文章时，也感觉很荒谬，不过当您看下去之后，会发现其实并非天马行空之谈。在文章开始之前我要先声明：本文只是做引导说明，若您读罢之后让没有“中奖”，只能说明您运气略差，本文不负任何责任。

新媒体抽奖活动，主要集中在微博、微信、贴吧这些新媒体平台以及app中。其本质都是通过活动促进文章或平台的阅读量或关注度，其活动一般分为两类：

使用第三方平台

使用第三方平台相对公平些，如在微博中是 转发抽奖平台，只要满足要求就能在抽奖平台中。其抽中的人是随机产生，因此这个只能看运气了。

转发抽奖平台 虽然公平公正公开，但也有其弊端。

1.舆论不可控，每一个参与的只要满足要求就能参与，随便发点什么就可以参加，账号的主体无法控制评论的导向。
2.人数限制，只有满足条件才能参与，只统计pv数据，与次数无关。
3.流程复杂，这类活动要经过备案审核，通过后才能发布。

注：转发抽奖平台使用流程
使用流程大致分为2步：导入和抽奖

导入

分为从地址导入和从微博导入。在从微博导入中选取相应微博，设定结束时间即可（结束时间可再次修改）。

抽奖

在抽奖界面可以设定一些过滤条件，比如@好友数、包含字词等等，当然，不要想着以这些作为内定，你设置的每个条件都会显示在结果页，增加条件或与条件不符小心被举报哦！

人为抽取

相比较第三方的机器抽取，人为抽取就是主打感情牌了，因此就有了作弊的嫌疑，在此来说说，曾经是怎么“作弊”的。

长短不一，脱颖而出

在”考试界”有这样一句话：三长一短选最短，三短一长选最长。说的也就是这个道理。与众不同的往往能在同类中脱颖而出，而字数是最简单的方式。当然，一般来看都是”三短一长”，所以，准备一个长评论，抓住小编的眼球吧。

活用表情，增加亮点

道理和第一点差不多，表情是个好东西，在文字中能够特别显眼，当然，说的是表情不是表情包。。。适当的加几个表情会让你的评论”脱颖而出”。

符合要求，最低底线

有些活动是有要求的，比如 @三位好友（三位好友真是个大忙人），有的要求关注XXX，有的还有时间限制。如果人为的也许会有漏洞，使用抽奖平台岂不是连凑分母的资格都没有了。看好要求，做好凑分母的第一步。

表达想法，获取同情

这是我的一次经历。是《名侦探柯南》电影上映时候送周边，万达影院做了一个分享朋友圈截图的活动。由于微信的封闭性，公众号的管理者只能看到后台统计的相关信息，而无法获取是谁分享，这就是这一抽奖方式的弊端。因此，在发送截图之后，又发了一大段对柯南喜爱的话，表达了自己无比渴求这一奖品。结果在开奖时幸运的中了。人心都是肉长的，小编想必在这么多内容中选出几条也是有选择恐惧症的，为什么不挑选更“期盼”的呢？

昵称头像，做好辅助

昵称和头像虽然是固定的，但也组成微博或微信评论中的一部分。举个例子，若一个以某明星签名照为奖品的微博下面，叫“爱豆XX（某明星）”的概率岂不是更大，抓住运营者的心理概率会更大。

寻找发现，尽力而为

想必你应该知道自己的人品是怎么样的，对于我我是感觉不好。所以类似于万中挑一赠手机，或者几百万中挑一赠汽车这些都悄悄路过。当然，这些活动都是会通过备案并通过第三方平台筛选的。若你幸运的中了，当然要恭喜恭喜，不过这种活动税费应该也不少交吧。

以上是我们评论的方式，目的是吸引运营者的眼球。而新媒体运营亦是如此，每一个运营者都是在吸引他人的眼球，以获得更高的浏览量和关注量，所以以上内容应该也会给运营工作者在微博微信等新媒体的运营中有所作用。

最后祝大家都能人品爆发，中中中大奖，如果没中，当我没说。
